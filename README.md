# Простая форма аутентификации на React

Этот проект был разработан на основе [Create React App](https://github.com/facebook/create-react-app).
Дополнительно были добавлены formik и yup для простой валидации

 Форма доступна по этому [адресу gh-pages](https://anton-sn.github.io/auth-form/).

Для деплоя необходимо сначала собрать билд, затем выполнить команду ```yarn deploy```
